# Task 1 
## Поиск подстроки префикс-функцией

Найдите все вхождения шаблона в строку, используя префикс-функцию

Длина шаблона – p, длина строки – n. 

Время O(n + p), доп. память – O(p), p ≤ 30000, n ≤ 300000. 

### Формат ввода

Шаблон, символ перевода строки, строка.

### Формат вывода

Позиции вхождения шаблона в строке.

### Пример входных данных 
```
a

aaa
```

### Пример выходных данных
```
0 1 2
```
